# Retail_Data_Analysis
The report analyzes sales, profit, discounts, shipping costs, customer segments, regions, product categories, and daily sales trends. Interactive filters allow users to explore performance by year, customer segment, and product container, helping identify top-performing products, regions, and business segments for better decision-making.
# Retail Market Analysis Dashboard (2010–2013)
This dashboard provides a comprehensive analysis of retail sales, profit, discounts, shipping costs, customer segments, and regional performance over the last four years. It helps identify sales trends, profit growth, customer behavior, and business performance across different product categories and regions.
## Tools Used
 * Excel Dashboard
 * Pivot Tables
 * Pivot Charts
 * Slicers
 * Data Visualization
